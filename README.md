# SQL Data Analysis

## Overview

This project focuses on **SQL Data Analysis**, where business insights are extracted from data using SQL concepts such as filtering, sorting, grouping, and aggregation. The project demonstrates how raw data can be transformed into meaningful information that supports data-driven decision-making.

Using Python and Pandas, SQL operations such as **SELECT, WHERE, ORDER BY, GROUP BY, COUNT, SUM, and AVG** are performed to analyze business performance and generate actionable insights.

---

## Objective

The objective of this project is to analyze the dataset using SQL-style queries and generate meaningful business insights through filtering, grouping, and aggregation techniques.

## How to Open and Run the Project

### Prerequisites

Make sure Python is installed on your system.

Download Python:

https://www.python.org/downloads/

While installing, enable:

✓ Add Python to PATH

---

### Install Required Libraries

Open Command Prompt (Windows) or Terminal (Mac/Linux) and run:

```bash
pip install pandas numpy openpyxl jupyter
```

---

### Open the Project

1. Download the notebook file (`.ipynb`).
2. Download the dataset file (`.xlsx`).
3. Place both files in the same folder.

---

### Start Jupyter Notebook

Open Command Prompt in the project folder and run:

```bash
jupyter notebook
```

This will open Jupyter Notebook in your browser.

---

### Run the Notebook

Execute cells one by one using:

```text
Shift + Enter
```

OR

Select:

```text
Run → Run All Cells
```

---

### Using Google Colab (Alternative)

1. Open Google Colab:

https://colab.research.google.com/

2. Upload the notebook file (`.ipynb`).

3. Upload the dataset file (`.xlsx`).

4. Run all cells.
